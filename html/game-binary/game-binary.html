💡 1. O que é um Bit?

Computadores não entendem "5" ou "A". Eles usam transistores que só têm dois estados: Ligado (1) ou Desligado (0). Cada dígito desse é um bit (Binary Digit).

Tempo de leitura: 30s
🧮 2. O Segredo dos Pesos

No nosso sistema decimal, as casas valem $1, 10, 100...$ No binário, a base é 2, então os pesos dobram de valor a cada casa da direita para a esquerda:

8 ➔ 4 ➔ 2 ➔ 1
Tempo de leitura: 45s
🚀 3. Como Converter?

Para formar um número, basta "ligar" (1) os pesos necessários para somar o valor desejado.

Exemplo: Para fazer o 5, precisamos de $4 + 1$. Ligamos o peso 4 e o peso 1: 0101

Tempo de leitura: 45s
