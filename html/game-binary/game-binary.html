💡 1. O que é um Bit?

Computadores usam transístores que só têm dois estados: Ligado (1) ou Desligado (0). Cada dígito é um bit.

🧮 2. O Segredo dos Pesos

No binário, a base é 2, então os pesos dobram de valor a cada casa da direita para a esquerda:

8 ➔ 4 ➔ 2 ➔ 1
🚀 3. Como Converter?

Para formar um número, basta "ligar" (1) os pesos necessários.
Exemplo (5): $4 + 1$ ligados = 0101
